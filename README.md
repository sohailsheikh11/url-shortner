# 🚀 url-shortner

A backend-powered URL shortener built with Node.js and MongoDB that generates unique short links and efficiently redirects users to original URLs.

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/sohailsheikh11/url-shortner
cd url-shortner
```

### 2. Install dependencies

**For backend**

```
cd backend
npm install
```

**For frontend**

```
cd frontend
npm install
```

---

### 3. Setup environment variables

Create a `.env` file in the `server` folder and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

### 4. Run the project

**Backend**

```
cd backend
npm run dev
```

**Frontend**

```
cd frontend
npm start
```
